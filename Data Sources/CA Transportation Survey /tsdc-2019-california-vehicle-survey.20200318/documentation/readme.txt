TSDC download folder structure:

|-- <study_name>_tsdc_download
          |-- data: folder containing the cleansed survey data .csvs
          |-- drive_cycles: folder containing the drive cycle data
          |-- documentation: folder containing documentation to come with web download. 
		(If the provider sends documentation, this will be inside. If not, TSDC generated docs will be inplace instead)
                  |--TSDC data dictionary
		  |--TSDC codebook
                  |--TSDC drive cycle routine docs
		 -OR-
		  |-- provider data dictionary, reports, etc
		 -AND-
	  	  |-- changelog.txt: date, generated metadata, and column changes/redactions/additions


Standard TSDC Cititation:
"Transportation Secure Data Center." ([YEAR]). National Renewable Energy Laboratory. Accessed [DATE]: www.nrel.gov/tsdc.



Questions? Email tsdc@nrel.gov.
